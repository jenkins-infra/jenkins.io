The GitHub checks tab includes a link to the ci.jenkins.io job that evaluates the pull request.  Open the GitHub checks tab and confirm that you can see the ci.jenkins.io link

image::/images/image2.png[Use parent pom, width=700px]
The ci.jenkins.io job evaluates Java 8 and Java 11 and  evaluates Windows and Linux.  It is a good way to confirm that your change is ready for review by others.
