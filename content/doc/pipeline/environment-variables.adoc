Environment variables are set using name/value pairs:

----
pipeline {
    environment {
        FOO = "BAR"
    }

    agent label:"master"

    stages {
        stage("foo") {
            sh 'echo "FOO is $FOO"'
        }
    }
}
----

Can add many more variables one per line like so:

----
    environment {
        FOO = "BAR"
        BAZ = "bang"
    }

----
