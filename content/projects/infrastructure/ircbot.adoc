= IRC Bot (Deprecated)

NOTE: The IRC bot is no longer in use. Jenkins has migrated to https://matrix.org[Matrix] for community chat and notifications. See the link:/projects/infrastructure/#chat[Infrastructure page] for current communication channels.

== Historical Reference

The Jenkins project previously used an IRC bot (`jenkins-admin`) to manage automation tasks in IRC channels, including:

* Managing user permissions in IRC channels
* Sending build notifications
* Handling administrative commands

This functionality has been superseded by the migration to Matrix.

== Matrix (Current)

Jenkins community communication now happens via Matrix.
To get started, visit https://www.jenkins.io/chat/[jenkins.io/chat].
